# BiliTimer

一个应用于Bilibili平台的篡改猴脚本平台的篡改猴脚本（Tampermonkey UserScript）。

为B站视频添加定时暂停功能。

## ✅ 功能

- 为B站视频添加定时暂停功能

  😏

## 🔍 安装方式
- 安装 [Tampermonkey 插件](https://www.tampermonkey.net/)
- 点击此处[安装脚本](https://greasyfork.org/zh-CN/scripts/559016-bilitimer) 或 [下载](./BiliTimer.js)并手动导入
